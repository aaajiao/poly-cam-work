# 项目宣言 (Manifesto)

[**文档中心**](README.md) | [**English Root README**](../README.md)

`poly.cam.work` 是艺术家 aaajiao 长期项目 *Symbiosis*（共生）的数字化档案与研究平台。

## 核心立场

本项目旨在通过 3D 扫描、点云渲染与富媒体标注，构建一个连接物理劳动与数字表现的桥梁。它不仅是一个技术工具，更是一个关于劳动、物质生产与全球贸易交织关系的数字化叙事空间。

## 核心概念：入神与出神

本项目的核心是两种精神状态的对立与呼应：

### 入神 (Absorption)
- **主体**: 亚克力工厂的师傅们。
- **状态**: 专注、沉浸、与材料/工艺合一。
- **意涵**: 劳动即冥想，专注即抵抗。这种状态与艺术家的创作心流高度相似，是通过制作在不确定性中获得意义的方式。

### 出神 (Trance)
- **主体**: 中国的年轻一代（派对青年）。
- **状态**: 迷失、漂浮、精神游离。
- **意涵**: 在社会转型与全球化浪潮中呈现的精神无根状态，通过遗忘或逃避来应对现实的生存策略。

## 数字化档案的意义

在 `poly.cam.work` 中，LiDAR 扫描不仅是几何数据的集合，更是对劳动现场、物质残骸与社会变迁的数字化见证。

- **记录不可见的劳动**: 将制造者从全球流通的廉价商品背后提取出来，赋予其可见性。
- **构建研究的缝隙**: 在算法治理与地缘政治的宏大叙事中，寻找人与材料、人与人之间真实的连接。
- **持续进化的共生**: 作为一个动态的平台，它将随着 *Symbiosis* 项目的深入而不断演进，记录下这十年乃至更长时间的物质与数字轨迹。
