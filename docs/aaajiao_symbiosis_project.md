## 一、项目概述

**Symbiosis（共生）** 是 aaajiao 正在进行的长期项目，探索中国制造业转型十年间的人与劳动、精神状态与全球政治经济的交织关系。

| 项目信息 | |
|----------|---|
| **艺术家** | aaajiao (徐文恺) |
| **时间跨度** | 2015年至今（十年合作 + 五年工作室共处） |
| **地点** | 上海亚克力工厂 |
| **媒介** | 纪录影像、数据可视化、电子游戏、亚克力雕塑 |

## 二、核心背景

### 十年共生关系

aaajiao 与上海一家亚克力工厂的合作始于2015年。工厂的师傅们参与制作 aaajiao 的雕塑作品。这种合作关系持续了**十年**。

**近五年（2020-2025）**，aaajiao 将他的上海工作室搬入这家亚克力工厂内部，与工人们共享同一个空间。这不是隐喻——是物理意义上的共处：艺术家在工厂内创作，工人在同一空间内劳动。

### 中国制造业转型的十年

这十年也是中国制造业剧烈转型的十年：

| 时期 | 现象 |
|------|------|
| **2015-2019** | 制造业升级，部分工人开始失业 |
| **2020-2023** | 疫情封控，供应链重组，大量工人失业 |
| **2023-2025** | 青年失业率飙升（一度超过20%），数据统计方法争议 |

与此同时，中国通过 Shein、Temu 等平台向全球输出**超低价商品**——利用"De Minimis"免税规则（欧盟€150以下免关税）。这是一种**"通缩输出"**，被视为贸易战争的一种形式。

**核心悖论**：国内工人大量失业 ↔ 国家持续向全球输出廉价商品

## 三、核心概念：入神与出神

项目的核心是两种精神状态的对立与呼应：

### 入神（Absorption）

| | |
|---|---|
| **主体** | 亚克力工厂的师傅们 |
| **状态** | 专注、沉浸、与材料/工艺合一 |
| **特征** | 手与机器同步，意识融入工艺，近乎冥想的劳动状态 |
| **意涵** | 这种专注与艺术家的创作状态高度相似——**劳动即冥想，专注即抵抗** |

> 当工人操作机器时，他们进入一种**全然沉浸**的状态——手与材料同步，意识溶解于工艺。这与艺术家的创作心流（creative flow）惊人地相似。这是通过**制作**获得意义的方式，是在不确定性中的**通过专注进行抵抗**。

### 出神（Trance）

| | |
|---|---|
| **主体** | 中国的年轻一代（派对青年） |
| **状态** | 迷失、漂浮、精神游离 |
| **特征** | 在派对/Club文化中成长，遭遇社会转型后呈现精神上的无根状态 |
| **意涵** | 通过**遗忘/逃避**来应对现实，一种被动的生存策略 |

> 在同一座城市里，很多年轻人在派对文化下成长。但当中国社会进入转型期，这些年轻人呈现出一种非常具体的**迷失的精神状态**。当一家上海传奇夜店关闭时，aaajiao 记录了它的最后几夜——身体在移动，但灵魂已经在别处。

### 两种状态的关系

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   入神 (Absorption)              出神 (Trance)              │
│   ─────────────────              ─────────────              │
│                                                             │
│   工厂师傅                        派对青年                   │
│   专注于工艺                      迷失于现实                 │
│   通过制作获得意义                通过遗忘逃避现实           │
│   主动的抵抗                      被动的漂流                 │
│                                                             │
│                    ↓          ↓                             │
│                                                             │
│              两种生存策略，同一种困境                         │
│              ──────────────────────────                     │
│              不确定性时代下的人类状态                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**入神**与**出神**是同一种困境的两面——都是在**不确定性（precarity）**下的生存策略。一个通过制作找到意义；另一个通过遗忘来逃避。

## 四、核心冲突

### 制造者 vs 地缘政治力量

项目揭示的核心冲突是**结构性的**：

| 层面 | 冲突 |
|------|------|
| **个体 vs 系统** | 工人的命运 vs 国家的贸易战略 |
| **劳动价值** | 手工艺的尊严 vs 被标签为"廉价商品" |
| **可见性** | 制造者被消隐 vs 商品的全球流通 |
| **政治经济** | 国内失业 vs 通缩输出 |

> 工厂的工人不仅仅是工匠——他们是贸易战争中**不知情的士兵**。在这十年里，aaajiao 目睹了大量裁员。然而与此同时，Shein 和 Temu 等平台向欧洲倾销超低价产品。这是**通缩作为武器**：失去工作的工人，正是让中国能够向全球输出经济压力的同一批人。

### 不可调和的现实

这种冲突是**不可改变的现实政治**：

- 低廉商品的制造者
- 被国家权力塑造成贸易战争的工具
- 个体无法逃脱的结构性命运

## 六、研究内容

### A Generation on Pause? 一代人的暂停？

项目包含对中国青年危机的深度研究：

#### The Climb & The Circuit Breaker (2020-2023)

> 2020年初，中国青年失业率约为13.8%，尽管受到疫情初期冲击，仍在可控范围内。然而，随着长期封控，情况急剧恶化...

#### Return with a New Metric (2024-2025)

> 在暂停六个月后，数据于2024年初恢复发布，但有一个重大变化：**学生被排除在统计之外**。理论上，这应该会大幅降低失业率...

#### The "Involution" Effect 内卷效应

> 数据揭示了令人不安的关联。当2022-2023年失业率飙升时，学术研究报告显示自残和抑郁症状也相应激增...

#### Methodology & Truth 方法论与真相

> 避免"黑箱"：2023-2025年的官方自杀数据稀缺。为了可视化这一点，我们使用了学术代理数据——学校调查的元分析...

### Trade Background 贸易背景

> 近年来，利用"De Minimis"豁免——价值低于€150的包裹免征关税——像 Shein 和 Temu 这样的中国电商平台实现了爆发式增长...

## 九、与 aaajiao 整体实践的关系

**Symbiosis** 是 aaajiao 艺术实践的**物质轨迹**，与其**数字轨迹**形成双螺旋：

| 数字轨迹 | 物质轨迹 |
|----------|----------|
| User → Bot → Player → Void-Seeker | 十年工厂合作 |
| Internet Void（数字缝隙） | 入神（Absorption） |
| amnesia / aphasia | 出神（Trance） |
| 算法治理批判 | 政治经济学批判 |
| Pinky Swear, Guard I..., Minority Algorithm | Symbiosis, Two Rituals, 1bit |

**Internet Void**（aaajiao 一直在寻找的"算法系统中的缝隙"）在物质世界中的对应是什么？

答案是：**入神**——工人专注于工艺时的状态。这是另一种"缝隙"，是在全球化生产链的碾压下，依然存在的**人与材料、人与人之间的连接**。
